Environment Variables

These are the essential variables you need to set in your `.env` file or platform dashboard:

SESSION_ID
Your session ID from the generator (required)
OWNER_NUMBER
Your WhatsApp number (e.g., 255618558502) – no + sign
BOT_NAME
Your bot's display name (default: iamlegendv2)
PREFIXES
Command prefixes (default: .,!,/)
COMMAND_MODE
public or private (default: public)
TIMEZONE
Your timezone (default: Africa/Nairobi)

Termux (Android)

5 minutes

Run the bot directly on your Android phone. Copy and paste these commands one by one:

pkg update && pkg upgrade -y
pkg install git nodejs ffmpeg libvips -y
git clone https://github.com/YOUR_USERNAME/iamlegendv2.git
cd iamlegendv2
npm install
cp sample.env .env
nano .env # Paste your SESSION_ID and OWNER_NUMBER here
npm start

To keep bot running after closing Termux:

pkg install tmux
tmux new -s bot
npm start
# Press Ctrl+B then D to detach

VPS / Linux Server

5-7 minutes

For Ubuntu/Debian servers – recommended for 24/7 operation:

curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt install -y nodejs git ffmpeg libvips-dev libwebp-dev
git clone https://github.com/YOUR_USERNAME/iamlegendv2.git
cd iamlegendv2
npm install
cp sample.env .env
nano .env # Add your SESSION_ID and OWNER_NUMBER
npm install -g pm2
pm2 start index.js --name iamlegendv2
pm2 save && pm2 startup

Troubleshooting

  • 🔍 Session ID not working? Make sure you copied the full string including the prefix Stanytz378/iamlegendv2_
  • 🔄 Bot disconnects? Check your internet connection. On cloud platforms, ensure you're not exceeding free tier limits.
  • 📝 "Cannot find module" error? Run npm install again and make sure all dependencies are installed.
  • 🚫 Commands not responding? Verify your OWNER_NUMBER in .env is correct and matches your WhatsApp number.
  • 💾 Data lost after restart? Add a database (MongoDB, PostgreSQL, etc.) for persistent storage.
